BTR Navbar

BlogTalkRadio Navigation bar

To start, BTR navbars are static (not fixed to the top) and include support for basic navigation. Place one anywhere within a .container, which sets the width of your site and content.

To make the BTR navbar fixed add one of the following classes: .navbar-fixed-top or .navbar-fixed-bottom.
